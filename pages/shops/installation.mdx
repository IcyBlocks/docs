## Installation

import { Steps } from 'nextra/components'
import { FileTree } from 'nextra/components'

<Steps>
### Install The Resources
 
Install the [**dependencies**](../shops#dependencies) <br/>
Install cubic_shops `[Found In Your` [**Keymaster**](https://keymaster.fivem.net) `Account as "Cubic 24/7"]`

### Configure Ox Inventory

<FileTree>
  <FileTree.Folder name="ox_inventory" defaultOpen>
    <FileTree.Folder name="data" defaultOpen>
      <FileTree.File name="items.lua" />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

```lua filename="items.lua" copy showLineNumbers
['hacking_device'] = { 
	label = "Hacking Device",
	weight = 200,
	stack = true,
	close = true,
	client = {
		event = 'cubic-storerobbery:client:vault:useitem',
	},
},
```

### Configure server.cfg

Make sure to start the resources in the correct order

```ini filename="server.cfg" copy showLineNumbers
ensure oxmysql
ensure ox_lib
ensure [framework] ## Note: A framework isn't required to run cubic_shops
ensure ox_inventory
ensure peuren_minigames
ensure cubic_shops
```

> Note: The SQL table is automatically added to your database
</Steps>
